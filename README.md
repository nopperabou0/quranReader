# Quran Reader

## Description
Quran reader using Raylib and C99

## Installation

build with 
```zig
zig build
```
 and run it with 
```zig
./zig-out/bin/quranReader-linux 
```
or 
```zig
./zig-out/bin/quranReader-windows.exe
```

alternatively for linux just run 

```zig
zig build run
```
