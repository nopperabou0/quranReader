# quranReader

## Description
Quran reader using raylib

## Installation

build with zig build and run it with 
```zig
./zig-out/bin/quranReader-linux 
```
            or 
```zig
./zig-out/bin/quranReader-windows.exe
```

alternatively for linux just run 

```zig
zig build run
```
